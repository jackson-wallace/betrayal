## Technologies Used

- **Go**: Backend server implementation.
- **TypeScript**: Frontend development for type-safe JavaScript.

## Getting Started

To set up the project locally, follow these steps:

1. **Clone the repository**:

   ```bash
   git clone https://github.com/jackson-wallace/betrayal.git
   ```

2. **Navigate to the project directory:**

   ```bash
   cd betrayal
   ```

3. **Make sure you have Go and TypeScript installed on your machine**

4. **Transpile .ts files to .js**

   ```bash
   tsc
   ```

5. **Install required Go modules**

   ```bash
   go mod tidy
   ```

6. **Run backend**

   ```bash
   go run *.go
   ```

7. **Navigate to** `localhost:8080` **in your browser**
